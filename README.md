# 浙大健康打卡定时自动脚本

 - 可定时，默认为每天0点开始执行
 - 默认每次提交上次所提交的内容（只有时间部分更新）
 - 系统表单如有更新，在当天自行手机打卡，后面会自动按照你更新后的选项继续打卡
 - 使用Github Actions完成自动化任务

 项目用于学习交流，仅用于各项无异常时打卡，如有身体不适等情况还请自行如实打卡~


> 感谢[Tishacy](https://github.com/Tishacy/ZJU-nCov-Hitcarder)同学

## Usage

1. Fork 本项目
2. 添加Secrets
3. 开启Actions
